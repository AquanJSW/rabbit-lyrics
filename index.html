Introduction

Rabbit Lyrics is an audio and timed lyrics synchronizer for web. No jQuery required. No dependencies.

Install?

You can install through NPM or download from here. 

npm install rabbit-lyrics --save

To import it into your project, you can either use Webpack way (require sass-loader):

import RabbitLyrics from 'rabbit-lyrics'

or link files in HTML:

<link href="node_modules/rabbit-lyrics/dist/rabbit-lyrics.css" rel="stylesheet" type="text/css"/>
<script src="node_modules/rabbit-lyrics/dist/rabbit-lyrics.js" type="text/javascript"></script>

Initialize

HTML

<audio id="a-little-song" src="..." controls></audio>

<div class="rabbit-lyrics" data-media="#a-little-song">
[00:05.00] This is a little song
[00:05.00] The song that nobody knows
...
</div>

Element with class rabbit-lyrics will be initialized as lyrics. It will use media element specified by data-media attribute. If no data-media found, it will try to search first audio or video element before.

JavaScript

<div id="lyrics">
[00:05.00] This is a little song
[00:05.00] The song that nobody knows
...
</div>
new RabbitLyrics({element: document.findElementById('lyrics')})

If no mediaElement option provided, it will try to search first audio or video element before.
